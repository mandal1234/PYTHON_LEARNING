# Interview Readiness Checklist

Use this after Day 90.

You are not ready only because you finished reading. You are ready when you can
build, explain, and debug.

## Python Readiness

- [ ] I can explain variables, data types, lists, dictionaries, and loops.
- [ ] I can write functions with parameters and return values.
- [ ] I can explain classes, objects, self, attributes, and methods.
- [ ] I can handle errors using try/except.
- [ ] I can solve beginner coding problems without copying.

## SQL Readiness

- [ ] I can write SELECT, WHERE, ORDER BY, INSERT, UPDATE, DELETE.
- [ ] I can explain primary key and foreign key.
- [ ] I can write one JOIN query.
- [ ] I can use COUNT, SUM, AVG, and GROUP BY.
- [ ] I can design basic tables for the tiffin project.

## Django Readiness

- [ ] I can explain project vs app.
- [ ] I can explain request-response flow.
- [ ] I can create models and run migrations.
- [ ] I can use Django admin.
- [ ] I can write views, URLs, templates, and forms.
- [ ] I can explain Django ORM queries.
- [ ] I can add login and permissions.
- [ ] I can write basic tests.

## DRF Readiness

- [ ] I can explain what an API is.
- [ ] I can explain serializers.
- [ ] I can create API views or generic views.
- [ ] I can explain viewsets and routers.
- [ ] I can explain authentication and permissions.
- [ ] I can test API endpoints.

## Project Readiness

- [ ] My project runs locally.
- [ ] My README is clear.
- [ ] My project has screenshots.
- [ ] My project has at least 5 meaningful features.
- [ ] My project has API examples.
- [ ] My project has tests.
- [ ] I can explain my model design.
- [ ] I can explain one bug I fixed.
- [ ] I can explain one thing I would improve.

## Mock Interview Rule

Before applying seriously, practice these three answers daily:

1. "Tell me about yourself."
2. "Explain your Django project."
3. "What did you learn during your career break?"

## Honest Final Rule

If you cannot explain something simply, revise it.

If you can explain it, build it, and debug it, you are becoming interview-ready.
