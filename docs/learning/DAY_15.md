It should:

- Store menu items.
- Store customers.
- Store orders.
- Calculate totals.
- Find high-value orders.
- Save a report file.

## Interview Questions And Answers

---

### 🟦 QUESTION

## **How do you approach a Python problem?**

### 🟩 ANSWER

**Key Points**

- Understand the required input.
- Decide what output is needed.
- Break the problem into small steps.
- Write and test one part at a time.

**Interview Answer**

"I first understand the input and expected output. Then I break the problem
into smaller steps, write code for one step, test it, and continue."

---

### 🟦 QUESTION

## **How do you organize Python code?**

### 🟩 ANSWER

**Key Points**

- Use clear variable names.
- Use functions for repeated logic.
- Use classes when modeling real-world objects.
- Use modules when code grows across files.

**Interview Answer**

"I organize Python code using clear names, functions for reusable logic,
classes for real-world objects, and modules when the project grows."
