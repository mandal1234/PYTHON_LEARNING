## Interview Questions And Answers

---

### 🟦 QUESTION

## **What is a command line app?**

### 🟩 ANSWER

**Definition**

A command line app is a program that runs in the terminal and takes user input.

**Key Points**

- It does not need a web page.
- It helps practice logic.
- It can still solve real problems.

**Interview Answer**

"A command line app is a program that runs in the terminal. It takes input,
processes logic, and shows output."

---

### 🟦 QUESTION

## **Why build small apps while learning?**

### 🟩 ANSWER

**Key Points**

- Small apps connect many concepts together.
- They help practice problem solving.
- They prepare us for larger Django apps.

**Interview Answer**

"Small apps help me combine concepts like functions, loops, lists, and
conditions into one working program."
